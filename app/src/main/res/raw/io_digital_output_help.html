I/O DIGITAL OUTPUT

The Demo

The BBC micro:bit has a metallic, brass coloured edge connector with a number of separate connection points known as "pins" plus a 3 volt connector and a GND connector for powering other devices. The other pins are for data and each may be used for either input or output and for digital values (0 or 1) or analogue values (0 - 255).

The IO Digital Output demo is designed to allow you to connect something suitable like an LED to data pin 0 and then from this application, using the great big on-screen button, you can switch the LED off, all from your phone all over Bluetooth! This is tailor made for people who like to take it easy and let technology do all the hard work... like getting up off the sofa and walking across the room to flick a switch :-)

The Code

This demo uses the Bluetooth IO Pin Service which contains three characteristics.

The PIN Data characteristic has a value part which consists of a variable number of pairs of bytes with each pair consisting of a pin number and a pin value. Write to this characteristic and each physical pin will have its state changed according to the value state and the way you've configured it using the other two characteristics. You can enable notifications on this characteristic (although this is not done in the demo) and notifications will deliver Pin Number / Value pairs for those pins defined as input pins by the Pin IO Configuration characteristic and whose value when read differs from the last read of the pin.

The PIN AD Configuration characteristic lets you control whether a pin is to be used with digital or analogue values. It uses one bit per pin and a bit value of 0 means digital and a value of 1 means analogue.

The PIN IO Configuration characteristic similarly lets you configure each pin to be used for either input or output. A bit mask representation is also used with a 0 meaning output and a 1 means input.

All the code for this demo can be found in ui.IoDigitalOutputActivity.java except for the Bluetooth operations which are in bluetooth.BleAdapterService.
